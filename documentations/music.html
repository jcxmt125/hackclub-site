Documentations for Song Maker Plus songs

Song Maker Plus is a project based on Google's Song Maker experiment.
You can find some songs I've made using it here!
Below are usage instructions and mp3 demos.
(Quick note: I can't make this page accessible to deaf people! Sorry...)

To view or modify the projects, you'll have to load the songs in SM+!
Paste https://www.songmakerpl.us/tracks?url= into your browser, then paste in the raw github file link for the json file!
Then, remove the .json from the end of the link. It should look something like this: https://raw.githubusercontent.com/jcxmt125/side-projects/main/musicLab/tense/track
So the final constructed link would be: https://www.songmakerpl.us/tracks?url=https://raw.githubusercontent.com/jcxmt125/side-projects/main/musicLab/tense/track
