Usage instructions

Logging

In a terminal: execute webNowPlayingLogger.py.
It'll create a file wnplog.txt with the log of played media, checking every second.

Reading

In a terminal: execute wnpLogRead.py.
Drag and drop, or provide a path to the log file.
Select the option you'd like to execute.
The log file is also human-readable.
